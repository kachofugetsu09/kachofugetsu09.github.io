选择排序

从第一个元素到最后一个元素中选择出一个最小的元素，和第一个元素进行交换，然后从第二个元素到最后一个元素中选择出最小的，与第一个进行交换，
以此类推排列，比如213排列为123

冒泡排序

不断比较相邻的元素，将数值大的元素往后冒从而实现排序到达最后升序排列

插入排序

将第二个元素和第一个元素进行比较，如果第二个元素更小，那么第一个元素向后移动，第二个元素插入第一个元素的位置
第三个元素和第一个元素第二个元素进行比较，如果把他插入到合适的位置


数组[5, 2, 9, 3, 6]
第一次迭代：

i = 1, x = 2, j = 0，数组状态变为 [2, 5, 9, 3, 6]
第二次迭代：

i = 2, x = 9, j = 1，数组状态变为 [2, 5, 9, 3, 6]（不需要移动元素，9已在正确位置）
第三次迭代：

i = 3, x = 3, j = 2，数组状态变为 [2, 3, 5, 9, 6]（将3插入到正确位置）
第四次迭代：

i = 4, x = 6, j = 3，数组状态变为 [2, 3, 5, 6, 9]（将6插入到正确位置）
最后，数组已按升序排列：[2, 3, 5, 6, 9]。每次迭代都通过比较 x 和 a[j]，将 x 插入到已排序的子数组中的正确位置，直到整个数组都排好序为止。这就是插入排序的工作方式。

基数排序
