## 使用方法

1. 访问 https://fontawesome.com/icons
2. 搜索关键词（如 "database", "code", "memory"）
3. 找到喜欢的图标，复制类名（如 `fas fa-database`）
4. 在 README.md 中使用：

```yaml
---
name: "Redis实现"
icon: "fas fa-memory"
---
```

## 其他图标库的使用

如果想用其他图标库，需要在 HTML 中引入对应的 CSS：

### Lucide Icons
```html
<script src="https://unpkg.com/lucide@latest/dist/umd/lucide.js"></script>
```
